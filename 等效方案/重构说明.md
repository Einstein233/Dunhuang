# CSEE-Gen v2.0 重构说明

## 模块结构

```
等效方案/
├── config.py              # 配置模块 - 集中管理所有超参数
├── data_loader.py         # 数据读取与准备模块
├── experiment_mapper.py   # 降雨 - 日照实验方案等效映射模块
├── utils.py               # 工具函数模块
├── main.py                # 主程序入口
├── __init__.py            # 包初始化
└── data/
    └── processed_weather_data.csv
```

## 配置模块 (config.py)

所有可配置参数集中在此模块中定义：

```python
from config import ExperimentConfig

# 方式 1: 使用默认配置
config = ExperimentConfig()

# 方式 2: 自定义配置
config = ExperimentConfig(
    # 事件提取
    no_rain_threshold_hours=3.0,       # 3 小时无雨界限
    solar_accumulation_hours=48,       # 日照能量累计 48 小时

    # 实验循环
    solar_duration_hours=12.0,         # 日照阶段 12 小时
    dry_duration_hours=12.0,           # 干燥阶段 12 小时
    cycle_rain_duration_hours=1.0,     # 降雨基础时长 1 小时

    # 脉冲降雨
    pulse_rain_max_duration_min=30.0,  # 单次降雨最大 30 分钟

    # 降雨强度
    rain_intensity_gears=[20, 40, 60], # 三个降雨强度档位

    # 日照强度
    max_solar_intensity=1000,          # 最大日照强度 1000 W/m²

    # 温度计算
    daytime_start_hour=8,              # 白天开始时间 8 点
    daytime_end_hour=20                # 白天结束时间 20 点
)
```

## 使用方式

### 方式 1: 命令行运行

```bash
# 使用默认配置运行
python main.py

# 指定参数运行
python main.py --days 20 --input data/data.csv --output plan.json
```

### 方式 2: Python 代码调用

```python
from experiment_mapper import ExperimentPlanGenerator, ExperimentConfig

# 使用默认配置
generator = ExperimentPlanGenerator(
    csv_path="data/processed_weather_data.csv",
    target_days=10
)
plan = generator.generate()
plan.save("experiment_plan.json")

# 使用自定义配置
config = ExperimentConfig(
    solar_duration_hours=12.0,
    dry_duration_hours=12.0,
    pulse_rain_max_duration_min=30.0,
    rain_intensity_gears=[20, 40, 60]
)
generator = ExperimentPlanGenerator(
    csv_path="data/data.csv",
    target_days=10,
    config=config
)
plan = generator.generate()
plan.save("experiment_plan.json")
```

### 方式 3: 使用便捷函数

```python
from experiment_mapper import generate_experiment_plan

plan = generate_experiment_plan(
    csv_path="data/data.csv",
    target_days=10,
    output_path="experiment_plan.json"
)
```

## 温度设置说明

每个实验循环包含 3 个阶段，各阶段温度设置如下：

| 阶段 | 温度设置 | 说明 |
|------|----------|------|
| Rain | 降雨小时内温度中位数 | 模拟降雨时的实际温度 |
| Solar | 白天 (8-20 点) 温度中位数 | 模拟日照阶段的白天温度 |
| Dry_Baseline | 全时段温度中位数 | 模拟干燥阶段的基准温度 |

所有阶段使用相同的恒定风速：全时段平均风速。

## 超参数列表

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `NO_RAIN_THRESHOLD_HOURS` | 3.0 | 无雨界限时间（小时） |
| `SOLAR_ACCUMULATION_HOURS` | 48 | 日照能量累计时间（小时） |
| `SOLAR_DURATION_HOURS` | 12.0 | 日照阶段时长（小时） |
| `DRY_DURATION_HOURS` | 12.0 | 干燥阶段时长（小时） |
| `PULSE_RAIN_MAX_DURATION_MIN` | 30.0 | 单次降雨最大时长（分钟） |
| `RAIN_INTENSITY_GEARS` | [20, 40, 60] | 降雨强度档位 |
| `MAX_SOLAR_INTENSITY` | 1000 | 最大日照强度 (W/m²) |
| `DAYTIME_START_HOUR` | 8 | 白天开始小时 |
| `DAYTIME_END_HOUR` | 20 | 白天结束小时 |

## 核心类说明

### ExperimentPlanGenerator
实验方案生成器，整合所有模块：
- `_load_data()`: 加载天气数据
- `_extract_events()`: 提取降雨事件
- `_merge_events()`: 执行事件合并
- `_couple_energy()`: 执行能量耦合
- `_generate_steps()`: 生成实验步骤

### RainEventExtractor
降雨事件提取器，以 3 小时无雨为界限切分独立事件。

### EventMerger
事件合并器，基于槽位预算合并 V 值最小的相邻事件。

### EnergyCoupler
能量耦合器，将自然降雨事件映射为实验室可执行的等效事件。

### WeatherDataLoader
天气数据加载器，从 CSV 文件加载数据并计算气候统计。
