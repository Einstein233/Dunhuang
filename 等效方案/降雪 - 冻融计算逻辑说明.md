# 降雪 - 冻融循环等效计算逻辑说明

## 一、数据来源与单位

### 1.1 CSV 数据文件格式
```csv
time,temperature_2m,relativehumidity_2m,rain,snowfall,shortwave_radiation,windspeed_10m
2024-01-24T10:00,-10.7,48,0.0,0.07,41.0,1.49
```

**重要：`snowfall` 列的单位是 **cm**（厘米）**

| 字段 | 单位 | 说明 |
|------|------|------|
| temperature_2m | °C | 2 米气温 |
| relativehumidity_2m | % | 相对湿度 |
| rain | mm | 降雨量 |
| **snowfall** | **cm** | **降雪量（厘米）** |
| shortwave_radiation | W/m² | 短波辐射 |
| windspeed_10m | m/s | 10 米风速 |

### 1.2 示例数据解读
- CSV 中 `snowfall: 0.07` 表示 **0.07 cm = 0.7 mm**
- CSV 中 `snowfall: 0.14` 表示 **0.14 cm = 1.4 mm**
- 全年总降雪量 `2.66` 表示 **2.66 cm = 26.6 mm**

## 二、自然冻融损伤指标计算

### 2.1 冻融事件识别条件
一次完整的冻融循环定义：
1. **冻结启动**：温度 < -2.2°C（冻结阈值）
2. **融化阶段**：温度 > 1.2°C（融化阈值）
3. **再冻结**：温度再次 < -2.2°C

### 2.2 冻融损伤值计算公式

$$D = \\alpha \\times |\\Delta T| \\times (T_{max} - T_{thaw}) \\times (T_{freeze} - T_{min}) \\times t^{\\beta}$$

其中：
| 参数 | 符号 | 默认值 | 说明 |
|------|------|--------|------|
| 权重系数 | α | 1.0 | 冻融损伤权重系数 |
| 时间指数 | β | 0.5 | 时间效应指数（平方根效应） |
| 冻结阈值 | T_freeze | -2.2°C | 冻结启动温度 |
| 融化阈值 | T_thaw | 1.2°C | 融化启动温度 |
| 温差 | ΔT | - | T_max - T_min（绝对值） |
| 持续时间 | t | - | 循环持续时间（小时） |

### 2.3 计算示例

**示例 1：强冻融循环**
- T_min = -10.0°C, T_max = 5.0°C, t = 24.0 h
- ΔT = |5.0 - (-10.0)| = 15.0°C
- 冻结幅值 = (-2.2) - (-10.0) = 7.8°C
- 融化幅值 = 5.0 - 1.2 = 3.8°C
- 时间因子 = 24^0.5 ≈ 4.90
- **损伤值 D = 1.0 × 15.0 × 3.8 × 7.8 × 4.90 ≈ 2178.1**

**示例 2：中等冻融循环**
- T_min = -5.0°C, T_max = 3.0°C, t = 12.0 h
- ΔT = 8.0°C
- 冻结幅值 = (-2.2) - (-5.0) = 2.8°C
- 融化幅值 = 3.0 - 1.2 = 1.8°C
- 时间因子 = 12^0.5 ≈ 3.46
- **损伤值 D = 1.0 × 8.0 × 1.8 × 2.8 × 3.46 ≈ 139.7**

**示例 3：弱冻融循环（无损伤）**
- T_min = -2.0°C, T_max = 1.5°C, t = 6.0 h
- 冻结幅值 = (-2.2) - (-2.0) = -0.2 → **0.0**（max 函数）
- **损伤值 D = 0.0**（未超过冻结阈值）

## 三、实验室标准冻融循环设置

### 3.1 标准参数设置

| 参数 | 符号 | 值 | 说明 |
|------|------|-----|------|
| 最低温度 | T_min_lab | -18.0°C | 实验室可达最低温 |
| 最高温度 | T_max_lab | 5.0°C | 实验室可达最高温 |
| 冻结时长 | t_freeze | 2.5 h | 冻结阶段持续时间 |
| 融化时长 | t_thaw | 1.5 h | 融化阶段持续时间 |
| 总时长 | t_total | 4.0 h | 一次完整循环 |

### 3.2 实验室标准损伤值计算

- T_min = -18.0°C, T_max = 5.0°C, t = 4.0 h
- ΔT = |5.0 - (-18.0)| = 23.0°C
- 冻结幅值 = (-2.2) - (-18.0) = **15.8°C**
- 融化幅值 = 5.0 - 1.2 = **3.8°C**
- 时间因子 = 4.0^0.5 = **2.0**
- **实验室标准损伤值 D_lab = 1.0 × 23.0 × 3.8 × 15.8 × 2.0 ≈ 2761.8**

## 四、自然→实验室等效转换

### 4.1 等效转换公式

$$等效次数 = \\lceil \\frac{D_{natural}}{D_{lab}} \\rceil$$

其中：
- D_natural：自然冻融事件损伤值
- D_lab：实验室标准循环损伤值（2761.8）
- ⌈⌉：向上取整（保守原则）

### 4.2 转换示例

| 自然事件 | D_natural | 等效比 | 向上取整 |
|----------|-----------|--------|----------|
| 强冻融 | 7106.5 | 2.57 | **3 次** |
| 中等冻融 | 1500.0 | 0.54 | **1 次** |
| 弱冻融 | 100.0 | 0.04 | **1 次** |

### 4.3 为什么 66 次自然循环 → 80 次实验室循环？

**原因分析：**

1. **自然冻融累积损伤大**
   - 66 次自然循环的总损伤值：110728.9
   - 等效比 = 110728.9 / 2761.8 ≈ **40.09**
   - 向上取整后总次数 = **80 次**

2. **每次自然事件的等效**
   - 即使较小的自然事件（如 D=358）也会等效为 1 次实验室循环
   - 66 次自然事件中，许多事件的等效比在 0.01~0.5 之间
   - 向上取整后，每个都至少贡献 1 次

3. **保守原则**
   - 向上取整确保实验室损伤 ≥ 自然损伤
   - 避免低估自然环境的累积效应

**数学验证：**
```
总自然损伤 / 实验室标准损伤 = 110728.9 / 2761.8 ≈ 40.09
如果全部向下取整：约 40 次
如果全部向上取整：约 80 次（实际）
```

## 五、降雪量计算与分配

### 5.1 单位转换

| 步骤 | 计算 | 结果 |
|------|------|------|
| CSV 原始值 | 2.66 | **2.66 cm** |
| 转换为 mm | 2.66 × 10 | **26.6 mm** |

### 5.2 保守原则分配

**公式：**
$$单次降雪量 (mm) = \\lceil \\frac{总降雪量 (mm)}{降雪事件次数} \\rceil$$

**示例计算（6 次降雪事件）：**
```
总降雪量 = 2.66 cm = 26.6 mm
单次降雪量（原始）= 26.6 / 6 ≈ 4.43 mm
单次降雪量（向上取整）= ⌈4.43⌉ = 5 mm
实际总降雪量 = 5 × 6 = 30 mm
```

**保守性验证：**
- 原始总降雪量：26.6 mm
- 分配后总降雪量：30 mm
- **30 mm > 26.6 mm** ✓（满足保守原则）

### 5.3 降雪强度计算

**公式：**
$$降雪强度 (mm/h) = \\frac{单次降雪量 (mm)}{降雪时长 (h)}$$

**示例：**
- 单次降雪量 = 5 mm
- 降雪时长 = 60 min = 1 h
- **降雪强度 = 5 / 1 = 5 mm/h**

## 六、实验方案编排流程

```
┌─────────────────────────────────────────────────────────────┐
│                     实验方案编排流程                        │
├─────────────────────────────────────────────────────────────┤
│  1. 读取 CSV 数据（snowfall 单位：cm）                        │
│     ↓                                                       │
│  2. 提取降雪事件 → 统计降雪次数 N_snow、总降雪量 S_cm       │
│     ↓                                                       │
│  3. 识别自然冻融事件 → 计算每次损伤 D_i                     │
│     ↓                                                       │
│  4. 计算实验室标准损伤 D_lab                                │
│     ↓                                                       │
│  5. 等效转换：每次自然事件 → ceil(D_i / D_lab) 次实验室循环 │
│     ↓                                                       │
│  6. 降雪分配：单次降雪量 = ceil(S_cm × 10 / N_snow) mm      │
│     ↓                                                       │
│  7. 编排实验步骤：                                          │
│     for 每次降雪事件 i:                                     │
│       - 降雪阶段（雪花机 ON，60 min）                       │
│       - 冻融循环 × ceil(D_i / D_lab) 次                    │
│         - 冻结阶段（-18°C，150 min）                        │
│         - 融化阶段（5°C，90 min）                           │
└─────────────────────────────────────────────────────────────┘
```

## 七、关键参数配置表

### 7.1 冻融损伤计算参数
```python
FTC_DAMAGE_ALPHA = 1.0        # α: 权重系数
FTC_DAMAGE_BETA = 0.5         # β: 时间指数
FTC_FREEZE_THRESHOLD = -2.2   # 冻结阈值 (°C)
FTC_THAW_THRESHOLD = 1.2      # 融化阈值 (°C)
```

### 7.2 实验室标准参数
```python
LAB_FTC_TMIN = -18.0          # 最低温 (°C)
LAB_FTC_TMAX = 5.0            # 最高温 (°C)
LAB_FTC_FREEZE_HOURS = 2.5    # 冻结时长 (h)
LAB_FTC_THAW_HOURS = 1.5      # 融化时长 (h)
```

### 7.3 降雪阶段参数
```python
SNOW_DURATION_MIN = 60.0      # 降雪时长 (min)
SNOW_TEMP_MIN = -10.0         # 降雪最低温 (°C)
SNOW_TEMP_MAX = -2.0          # 降雪最高温 (°C)
```
