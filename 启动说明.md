# 启动说明

## 推荐方式

在项目根目录执行：

```bash
npm start
```

这会自动：

1. 启动 `sql-agent` 中的 Docker 服务
2. 启动 `WeatherVisualization`
3. 启动 `front_con`
4. 启动 `back_con`

## 启动前提

- 已安装 `Node.js`
- 已安装 `npm`
- 已安装并打开 `Docker Desktop`

## 第一次运行前

请先安装依赖：

```bash
npm install
cd WeatherVisualization && npm install
cd ../front_con && npm install
cd ../back_con && npm install
cd ../sql-agent && npm install
cd ..
```

## 访问地址

- `WeatherVisualization`: `http://127.0.0.1:5173`
- `front_con`: `http://127.0.0.1:8080`
- `back_con`: `http://127.0.0.1:3000`
- `sql-agent`: `http://127.0.0.1:3001`
- `MySQL`: `127.0.0.1:3308`

## 停止方式

在启动终端按：

```bash
Ctrl + C
```

## 手动启动方式

如果一键启动失败，可以分别执行：

```bash
cd sql-agent
docker compose up -d --build
```

```bash
cd back_con
npm run dev
```

```bash
cd front_con
npm run dev
```

```bash
cd WeatherVisualization
npm run dev
```
