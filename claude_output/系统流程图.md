# 气候数据到实验方案转换系统流程图

## 1. 整体系统流程

```mermaid
flowchart TD
    A[原始气候数据<br/>逐小时 CSV] --> B[数据预处理<br/>小时→日聚合]
    B --> C[季节分割<br/>冬季识别 + 非冬季三段切分]
    C --> D[季节范围映射<br/>日索引→小时索引]

    D --> E1[春季数据处理]
    D --> E2[夏季数据处理]
    D --> E3[秋季数据处理]
    D --> E4[冬季数据处理]

    E1 --> F1[降雨等效计算]
    E2 --> F2[降雨等效计算]
    E3 --> F3[降雨等效计算]
    E4 --> F4[降雨等效计算]

    E1 --> G1[日照等效计算]
    E2 --> G2[日照等效计算]
    E3 --> G3[日照等效计算]
    E4 --> G4[日照等效计算]

    E1 --> H1[环境因子等效<br/>温/湿/风]
    E2 --> H2[环境因子等效]
    E3 --> H3[环境因子等效]
    E4 --> H4[环境因子等效]

    F1 & G1 & H1 --> I1[春季实验方案]
    F2 & G2 & H2 --> I2[夏季实验方案]
    F3 & G3 & H3 --> I3[秋季实验方案]
    F4 & G4 & H4 --> I4[冬季实验方案]

    I1 & I2 & I3 & I4 --> J[最终输出<br/>JSON 格式实验方案]
```

## 2. 季节分割详细流程

```mermaid
flowchart TD
    Start[开始季节分割] --> Step1[读取逐日最低温序列 tmin]
    Step1 --> Step2[标记冷日 coldt = 1 if tmin < T0]

    Step2 --> Step3[构建冬季连续段<br/>允许暖间隙≤g 天]
    Step3 --> Step4[过滤短段<br/>长度 < min_len 剔除]

    Step4 --> Step5{检查跨年情况？}
    Step5 -->|年初+年末有冬季 | Step6a[合并为跨年冬季<br/>两段保留]
    Step5 -->|正常情况 | Step6b[选取最长冬季段]

    Step6a --> Step7[确定非冬季补集区间]
    Step6b --> Step7

    Step7 --> Step8[对非冬季区间执行 split]
    Step8 --> Step9[计算τ1: 春/夏分界]
    Step9 --> Step10[计算τ2: 夏/秋分界]

    Step10 --> Step11[输出：春/夏/秋/冬索引区间]
    Step11 --> End[分割完成]
```

## 3. split 函数详细流程（τ1/τ2 计算）

```mermaid
flowchart TD
    Start[开始 split] --> Input[输入：非冬季区间 tmeanp:q]
    Input --> Smooth[平滑温度序列<br/>移动平均 win=7]

    Smooth --> FindPeak[找到峰值点<br/>强制夏季包含最高温]

    FindPeak --> Tau1[计算τ1 春/夏分界]

    subgraph Tau1_Calc [τ1 计算]
        T1_1[计算春季参考温度<br/>前 min_seg_len 天中位数]
        T1_2[阈值 = 参考温度 + δ6.5°C]
        T1_3[找首次连续 run_need 天≥阈值]
        T1_4[起点 -1 = τ1]
    end

    Tau1 --> Tau1_Calc
    Tau1_Calc --> Tau1_Fallback{找到？}
    Tau1_Fallback -->|否 | Tau1_Fall[回退：最大上升斜率点]
    Tau1_Fallback -->|是 | Tau2[计算τ2 夏/秋分界]
    Tau1_Fall --> Tau2

    subgraph Tau2_Calc [τ2 计算]
        T2_1[计算夏季参考温度<br/>τ1+1 到 peak 中位数]
        T2_2[阈值 = 参考温度 - δ6.5°C]
        T2_3[找首次连续 run_need 天≤阈值]
        T2_4[起点 = τ2]
    end

    Tau2 --> Tau2_Calc
    Tau2_Calc --> Tau2_Fallback{找到？}
    Tau2_Fallback -->|否 | Tau2_Fall[回退：最陡下降点]
    Tau2_Fallback -->|是 | Output[输出：spring, summer, autumn]
    Tau2_Fall --> Output
    Output --> End[split 完成]
```

## 4. 降雨等效计算流程

```mermaid
flowchart TD
    Start[开始降雨等效计算] --> Step1[读取逐小时降雨序列]

    Step1 --> Step2[提取降雨事件<br/>合并间隔≤3h 的连续降雨]
    Step2 --> Step3[过滤小事件<br/>总雨量 < 0.3mm 剔除]

    Step3 --> Step4{事件数 ≥ 5?}
    Step4 -->|是 | Step5a[GMM 二分类聚类]
    Step4 -->|否 | Step5b[回退：按均值处理]

    Step5a --> Step6[计算推荐事件雨量<br/>maxlarge_mean, P75]
    Step5b --> Step6

    Step6 --> Step7[计算小/中/大雨阈值<br/>GMM 聚类或分位数回退]

    Step7 --> Step8[分工况计算实验参数]

    subgraph Condition_Calc [工况计算]
        C1[小雨工况]
        C2[中雨工况]
        C3[大雨工况]
    end

    Step8 --> Condition_Calc
    Condition_Calc --> Output[输出实验方案<br/>次数/时长/强度]
    Output --> End[计算完成]
```

## 5. 日照等效计算流程

```mermaid
flowchart TD
    Start[开始日照等效计算] --> Step1[检测降雨事件]

    Step1 --> Step2{有降雨事件？}
    Step2 -->|无 | NoRain[返回空结果]
    Step2 -->|有 | Step3[构建雨后窗口]

    Step3 --> Step4[窗口定义<br/>start = rain_end+1h<br/>end = minrain_end+Wmax, next_rain_start-1h]

    Step4 --> Step5[计算 Ecap<br/>窗口内短波辐射累加 MJ/m²]
    Step5 --> Step6[Ecap 统计<br/>P50/P75/均值/最大/最小]

    Step4 --> Step7[收集窗口辐射样本]
    Step7 --> Step8[过滤日光样本<br/>shortwave > daylight_threshold]
    Step8 --> Step9[计算代表强度<br/>I_rep =  percentile P90]

    Step6 & Step9 --> Output[输出日照等效参数]
    Output --> End[计算完成]
```

## 6. 环境因子等效计算流程

```mermaid
flowchart TD
    Start[开始环境因子等效] --> Step1[读取季节索引范围]

    Step1 --> Step2[提取季节内样本<br/>温度/湿度/风速]

    Step2 --> Step3{判断季节类型？}

    Step3 -->|春季 | T1[温度等效 = P50 分位数]
    Step3 -->|夏季 | T2[温度等效 = P75 分位数]
    Step3 -->|秋季 | T3[温度等效 = P50 分位数]
    Step3 -->|冬季 | T4[温度等效 = P25 分位数]

    T1 & T2 & T3 & T4 --> RH[湿度等效 = 季节均值]
    RH --> U[风速等效 = 季节均值]

    U --> Output[输出：T_equiv, RH_equiv, U_equiv]
    Output --> End[计算完成]
```

## 9. 关键决策点

```mermaid
flowchart TD
    D1{冬季是否跨年？}
    D1 -->|是 | D1a[保留两段冬季<br/>年初 + 年末]
    D1 -->|否 | D1b[单段冬季]

    D2{非冬季长度<br/>≥ 3*min_seg_len?}
    D2 -->|是 | D2a[正常切分三段]
    D2 -->|否 | D2b[整体视为夏季]

    D3{降雨事件数 ≥ 5?}
    D3 -->|是 | D3a[GMM 聚类]
    D3 -->|否 | D3b[均值 fallback]

    D4{GMM 某类为空？}
    D4 -->|是 | D4a[回退均值]
    D4 -->|否 | D4b[正常输出]

    D5{有降雨事件？}
    D5 -->|是 | D5a[计算日照等效]
    D5 -->|否 | D5b[返回空结果]

    D1a & D1b --> D2
    D2a & D2b --> D3
    D3a & D3b --> D4
    D4a & D4b --> D5
    D5a & D5b --> Done[完成]
```
